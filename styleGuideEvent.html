2. 반복객체 높이 조절

- ui.contAutoHeight(".test1 li",[[3],[2],[1]]);
- 파라미터순서 : 타겟, PC, 테블릿, 모바일 순서대로 반복 수 지정
- 반응형 포인트는 1024, 768 기본설정 변경이 필요한경우 ui.contAutoHeight(".test1 li",[[3, 1026],[2, 760],[1]]); 객체 다음 분기값을 설정할 수 있음

- 3개 포인트로 해결이 안될경우 직접 resize 스크립트 사용 (타겟, 반복객체 수)
- ui.contAutoSetion(".test1 li", "4"); 

		$(window).bind('resize', function () {
			//pc
			if(window.innerWidth >= 1041){
				- ui.contAutoSetion(".test1 li", "4");

			//테블릿
			else if(window.innerWidth >= 768){
